# 🌱 Dual-Track Pre-Symptomatic Pest & Disease Detection for Mango Orchards Using Multimodal AI

## 📌 Overview

This project presents an intelligent, IoT-based multimodal system designed to detect and predict pest and disease conditions in mango orchards at an early stage. The system focuses on identifying risks before severe damage occurs, thereby improving crop yield, reducing losses, and enabling precision agriculture practices.

The project follows a **dual-track approach**, combining visible symptom detection using image processing and environmental monitoring using IoT sensors.

---

## 🎯 Objectives

* To detect pest and disease conditions in mango orchards at an early stage
* To reduce yield loss caused by delayed detection
* To integrate environmental sensing with image-based analysis
* To develop a smart agriculture solution for real-time monitoring
* To assist farmers with timely alerts and decision-making

---

## ⚠️ Problem Statement

Mango orchards suffer from **40–45% annual yield loss** due to late detection of pests and diseases. Traditional monitoring methods rely on manual inspection, which is time-consuming and ineffective in early-stage detection.

There is a lack of systems that:

* Provide **pre-symptomatic detection**
* Combine **environmental and visual data**
* Deliver **real-time insights to farmers**

---

## 💡 Proposed Solution

This project introduces a **dual-track detection system**:

### 🔍 Track A (Our Work)

* Image-based disease detection using leaf, fruit, and panicle images
* Environmental monitoring using IoT sensors (temperature, humidity, moisture, etc.)
* Multimodal AI model combining image and sensor data

### 🧪 Track B (Complementary Work)

* Detection of invisible parameters using VOC (smell) sensors
* Identification of diseases before visible symptoms appear

---

## 🧠 System Architecture

The system consists of:

* IoT sensors for environmental data collection
* Image acquisition system for plant monitoring
* Data preprocessing and feature extraction
* Deep learning models (CNN + LSTM)
* Multimodal data fusion
* Mobile/web interface for alerts and monitoring
