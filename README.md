# PRE---SYMPTOMATIC-PEST-DISEASE-DETECTION-FOR-MANGO-ORCHARDS-USING-MULTIMODAL-AI
**Project Description (2 lines):**  A dual-track smart system that uses IoT-based environmental monitoring and image processing to detect and predict mango pest and disease conditions. It applies multimodal AI (CNN + LSTM) to enable early intervention, reduce yield loss, and improve precision agriculture.
